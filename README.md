# Latte Studio ☕

一个优雅的咖啡制作记录工具，帮助你追踪每一次咖啡制作的参数，优化你的咖啡制作流程。

## ✨ 功能特性

- 📝 **制作记录** - 记录粉重、研磨度、水温、牛奶重量、萃取时间、液重等关键参数
- 🏷️ **标签系统** - 使用标签标记风味、口感、流速等特征
- ⭐ **评分系统** - 为每次制作评分，标记"神之萃取"
- 📊 **历史记录** - 查看所有制作历史，支持筛选和搜索
- 🛠️ **资产管理** - 管理咖啡豆、咖啡机、磨豆机、牛奶等资产
- 💰 **成本计算** - 自动计算每次制作的单位成本
- 📱 **移动端优化** - 专为移动设备设计，支持全面屏适配
- 🌙 **暗色主题** - 护眼的暗色界面设计

## 🚀 使用方法

1. 直接在浏览器中打开 `index.html` 即可使用
2. 所有数据存储在浏览器本地（LocalStorage），无需服务器
3. 首次使用建议先在"资产"页面添加你的咖啡豆、设备等信息

## 📋 主要功能说明

### 记录页
- 选择咖啡豆和牛奶
- 设置咖啡机和磨豆机
- 调整制作参数（粉重、研磨度、水温等）
- 记录萃取时间和液重
- 添加标签和评分
- 保存制作记录

### 历史页
- 查看所有制作记录
- 按日期、评分、标签筛选
- 搜索历史记录
- 编辑或删除记录

### 资产页
- 管理咖啡豆（添加图片、烘焙日期、价格等）
- 管理咖啡机和磨豆机
- 管理牛奶品牌
- 调整咖啡豆剩余量

## 🛠️ 技术栈

- **HTML5** - 纯前端应用
- **Tailwind CSS** - 样式框架（CDN）
- **Lucide Icons** - 图标库（CDN）
- **LocalStorage** - 数据存储

## 📝 注意事项

- 所有数据存储在浏览器本地，清除浏览器数据会导致数据丢失
- 建议定期备份重要数据
- 支持现代浏览器（Chrome、Safari、Firefox、Edge 等）
